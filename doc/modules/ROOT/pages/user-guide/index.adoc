== Prerequisites

Before diving into the details, make sure you have:

* A {cpp26} compiler with P2996 support (Bloomberg Clang P2996 fork)
* Basic understanding of {cpp} templates and concepts
* Familiarity with memory layout concepts (size, alignment, padding)

== Quick Reference

=== Core Functions

[source,cpp]
----
get_layout_signature<T>()      // Get layout signature
get_layout_hash<T>()           // Get 64-bit hash
get_layout_verification<T>()   // Get dual-hash verification
signatures_match<T1, T2>()     // Compare layouts
is_serializable_v<T, P>        // Check serializability for platform set P
has_bitfields<T>()             // Check for bit-fields
----

=== Concepts

[source,cpp]
----
Serializable<T>                // Type is safe for binary serialization
LayoutCompatible<T, U>         // Types have same layout
LayoutMatch<T, Sig>            // Layout matches signature
LayoutHashMatch<T, Hash>       // Layout hash matches value
----

=== Macros

[source,cpp]
----
TYPELAYOUT_BIND(Type, Sig)     // Bind type to expected signature
----
