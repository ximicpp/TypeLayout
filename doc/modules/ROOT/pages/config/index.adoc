Configuration options and platform support for Boost.TypeLayout.

== Compiler Requirements

Boost.TypeLayout requires a compiler with {cpp26} and P2996 static reflection support.

| 19.0+

| GCC

| ⏳ Waiting for P2996 support

| -

== Quick Setup

=== CMake Configuration

[source,cmake]
----
cmake_minimum_required(VERSION 3.20)
project(my_project)

# Require C++26
set(CMAKE_CXX_STANDARD 26)
set(CMAKE_CXX_STANDARD_REQUIRED ON)

# Add reflection flag for Clang
if(CMAKE_CXX_COMPILER_ID MATCHES "Clang")
    add_compile_options(-freflection)
endif()

# Find Boost.TypeLayout
find_package(boost_typelayout CONFIG REQUIRED)

add_executable(my_app main.cpp)
target_link_libraries(my_app boost_typelayout)
----

=== Manual Compilation

[source,bash]
----
# Using Clang with P2996 support
clang++ -std=c++26 -freflection -I/path/to/boost/typelayout main.cpp -o my_app
----

== Feature Detection

Use feature test macros to check for required capabilities:

[source,cpp]
----
#if !defined(__cpp_reflection)
    #error "C++26 reflection (P2996) is required"
#endif

#if !defined(__cpp_constexpr) || __cpp_constexpr < 202211L
    #error "Full constexpr support is required"
#endif
----
